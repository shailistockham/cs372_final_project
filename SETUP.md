Here is everything you need to install and run the Euchre win predictor.

## What you need before starting.
Python 3.8 or higher, as the program is written in this programming language. 
Jupyter Notebook, as the program needs an interactive coding environment.

## Installation
Step 1: Download the Euchre_bot.ipynb file by cloning in Git or downloading and unzipping the file.
Step 2: Install the libraries from requirements.txt by running pip install -r requirements.txt in terminal for Mac/Linux or Command Prompt for Windows.
Step 3: Launch Jupyter Notebook and open the Euchre_bot file.
Step 4: Run the program either all at once or one cell at a time.
Note: You can also run the program in Google Colab by uplaoding the ipynb file. All libraries are pre-installed.
